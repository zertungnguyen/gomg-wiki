Search Runes

Filter 144 runes from RuneDataTable. Rarity + Material mirror the in-game Codex → Runes → Filter dialog. Tags is a wiki-side facet drawn from FilterTags_EN (~41 atomic effect tags). Pills within a tab OR; across tabs AND.

The Resonance column lists the full R1 / R3 / R5 / R7 ladder per rune — exactly the four lines the in-game Rune Detail panel prints, no pre-computed totals. Tiers unlock at stack thresholds 1 / 3 / 5 / 7 of same-material runes equipped across your team's 10 gear slots. R1 is always the rune's own effect verbatim. R3 and R7 normally pull from RuneMaterialDataTable (the per-material set bonuses), but for materials whose stored row reads "Base effect +N%" (Copper) the engine substitutes the rune's own description instead — so Copper R3/R5 also print verbatim, and Copper R7 prints with every numeric digit ×2. Material pill hover still shows the raw R3/R7 strings from the DataTable.
