How to use

  • Daily gacha: open the Gacha Optimizer, click your 7 tags, see best combo (3-star density %).
  • Unit research: browse Units sorted by step. Each profile lists active skill, race/trait/title/perk, talent pool (exclusive + race + class + element + generic).
  • Game term lookup: keywords like Chill, Toughness, Weakness exposed are linked from unit profiles to dedicated keyword pages.

Notes

  • Step = rarity: S = 3-star (highest), A = 2-star, B = 1-star, C = 0-star.
  • Alter vs Standard: alter units (suffix _NNN, e.g. M53301_001) are skinned variants with different element/map/kit. Treat as separate units in gacha optimizer.
  • Build outlooks for non-S units are intentionally not maintained.
  • Upcoming alters (loc-only, no DataTable row yet) are excluded from this publish.
